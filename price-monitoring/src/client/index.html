    Обновить текст из Excel

    Выберите файл Excel со столбцами Id и Description. Значения будут записаны в колонку text_avito таблицы VSE4 по совпадению ID.

    Файл не выбран

    Внести ImageUrls из Excel

    Выберите файл Excel со столбцами Id и ImageUrls. Значения будут записаны в колонку ImageUrls таблицы VSE4 по совпадению ID.

    Файл не выбран

    ЯД — ImageUrls

    Заполняет ImageUrls из link_foto только для строк где ImageUrls = null. Заменяет указанный префикс на yandex_disk://, затем все \ → /.

    Исправить десятичные разделители

    Проверка статусов объявлений

    Сравнивает статус объявления на Авито и в БД. Выберите файл Excel со списком активных объявлений Авито (столбец Id).
    Проверка 1 — объявления из файла, которые в БД помечены как removed.
    Проверка 2 — объявления в БД (активные), которых нет в файле Авито.

    Авито загрузка

    Формирование Excel-файла для массовой загрузки объявлений на Авито. Поля из БД подставляются автоматически, остальные берутся из настроек ниже.

    Статические поля

    Поля из БД (подставляются автоматически): Id, AvitoId, Title, Description, Price, ImageUrls, RimBolts, RimBoltsDiameter, RimBrand, RimDIA, RimDiameter, RimModel, RimOffset, RimType, RimWidth

    Замена фотографий

    Привязка фотографий из каталога к карточкам VSE4. После привязки фото загружаются в Cloudflare R2 и URL сохраняются в ImageUrls.
